# EcoRide
Application web de covoiturage écrite en PHP / MySQL / HTML / CSS

## Fonctionnalités principales
 - Recherche de trajets avec l'application des filtres (US3, US4)
 - Détail, participation et notation de trajets (US5, US6)
 - Géstion des utilisateurs, véhicules et préférences (US7 à US9)
 - Historique et annulation (US10)
 - Espace chauffeur / passager / employé / administrateur (US8 à US13)
 - Sécurité : requêtes préparées, CSRF, session sécurisée, .htaccess

## Installation
1. Importer 'script.sql' dans MySQL.
2. Configurer '/config/db.php' avec tes identifiants de base de données.
3. Lancer le site via 'http://localhost/'.
